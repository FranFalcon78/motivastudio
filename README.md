# CursoPlayer Web

CursoPlayer Web is a cross-platform, browser-based video practice tool
designed for dancers and movement-focused learners. It runs directly in
any modern browser and provides precise A--B looping, speed control,
mirroring, and segment analysis to help users study technique, improve
body awareness, and build long-term motor memory.

-   **Clean & Lightweight:** No installation required --- works

-   **Cross-Platform:** Compatible with Windows, Linux, macOS, Android,
